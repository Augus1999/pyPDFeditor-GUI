# pyPDFeditor-GUI
This project is based on PyQt5 and PyMuPDF

## Functions
* merge
* organise pages
* encrypte

## Install
double click [setup.py](https://github.com/Augus1999/pyPDFeditor-GUI/blob/main/setup.py) or run ```python setup.py``` to install all required packages
