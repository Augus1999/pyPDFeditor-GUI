# pyPDFeditor-GUI

This project is based on PyQt5 and PyMuPDF.

## Functions
* merge files
* organize pages
* save page as image
* extract image from page
* encrypt and add watermarks

## Install
Run `pip install -r requirements.txt` to install all required packages.
